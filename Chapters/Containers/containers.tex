\chapter{Containers}
\label{chap:containers}
\vspace{10.5cm}
The \href{https://en.cppreference.com/w/cpp/container}{Containers library} 
is a \highlightit{generic collection of class templates and algorithms} 
that allow programmers \highlightit{to easily implement 
common data structures} like queues, lists and stacks.
\newline
The container classes are:
\begin{itemize}
    \item \highlightit{sequence containers}
    \newline
    implement data structures which can be accessed sequentially.
    \item \highlightit{associative containers},
    \newline    
    can be searched in $O(\log{n})$.
    \item \highlightit{unordered associative containers}
    \newline
    can be searched in $O(1)$ on average and in $O(n)$ in the worst one.
\end{itemize}
